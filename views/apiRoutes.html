Hi , This is the Backend Part (REST API) of DSABook web application


Auth Related Endpoints

    POST /api/auth/signup/

    POST /api/auth/signin/

    POST /api/auth/signout/

Blog Related Endpoints

    GET /api/blogs/

    GET /api/blogs/:id/

    POST /api/blogs/

    PATCH /api/blogs/:id/

    DELETE /api/blogs/:id

User Related Endpoints

    GET /api/users/

    GET /api/users/:username/

    POST /api/users/:username/myblog
